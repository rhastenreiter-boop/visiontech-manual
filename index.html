Visiontech Exp — Guia de Uso
Manual da equipe de expedição · Março 2026
1. Acessando o sistema
1
Abra o navegador e acesse
Digite na barra de endereço: https://visiontech-exp.onrender.com
Recomendamos usar o Google Chrome para melhor experiência.
2
Aguarde carregar
Na primeira vez do dia o sistema pode demorar até 30 segundos para acordar. Aguarde e a tela carregará normalmente.
2. Entendendo a tela
🟣
Lista de Pré-faturamento — aparece no topo quando há pedidos aguardando liberação do financeiro. Use para se preparar com antecedência.
📋
Lista de Pré-faturamento (topo)
Mostra pedidos que ainda estão no financeiro mas já podem ser preparados. Clique em "Abrir e Imprimir" para ver a Ordem de Faturamento em PDF. O item ficará riscado após a impressão.
🧾
Coluna "A Faturar"
Pedidos liberados pelo financeiro e prontos para iniciar o processo de expedição.
📦
Coluna "Em Separação"
Pedidos que estão sendo separados no estoque.
✅
Coluna "Faturado"
Pedidos faturados aguardando envio para a transportadora.
3. Fluxo do pedido
🟣 Pré-faturamento
→
🧾 A Faturar
→
📦 Em Separação
→
✅ Faturado
→
🚚 Enviado
ℹ️
O sistema atualiza automaticamente a cada 30 segundos. Novos pedidos aparecem sem precisar recarregar a página.
4. Trabalhando com um pedido
1
Abra o PDF da Ordem de Faturamento
Clique no botão laranja "📄 Ordem de Faturamento" dentro do card para abrir o PDF no navegador.
2
Mude o status conforme o andamento
Use o campo Status dentro do card para indicar em qual etapa o pedido está.
3
Preencha NF e Rastreio ao enviar
Quando o pedido for enviado, preencha o número da NF e o código de rastreio antes de salvar como Enviado.
4
Clique em 💾 Salvar
O sistema atualiza o CRM automaticamente com o status, NF e rastreio. O card sairá do kanban ao ser marcado como Enviado.
5. Status disponíveis
🧾 A faturar 📦 Em separação ✅ Faturado 🚚 Enviado ⚠️ Voltar ao Comercial
⚠️
Voltar ao Comercial — use apenas quando houver um problema que impeça o andamento do pedido. O sistema pedirá uma confirmação antes de executar e o card retornará automaticamente para o time comercial no CRM.
6. Botões de cada card
BotãoO que faz
💾 Salvar Salva o status, NF e rastreio. Atualiza o CRM automaticamente.
⚠️ Reportar Problema Abre formulário para reportar um erro. Envia mensagem via WhatsApp direto para o vendedor responsável pelo pedido.
🗑️ Excluir card Remove o pedido do sistema. Use apenas se o card foi inserido por engano.
7. Como reportar um problema
1
Clique em "⚠️ Reportar Problema"
O botão amarelo fica abaixo do botão Salvar em cada card.
2
Escolha o tipo do problema
Selecione uma das opções: Erro no cadastro do cliente, Anexo incorreto ou ausente, Pedido duplicado ou Outro.
3
Descreva o problema (opcional)
Use o campo de observação para dar mais detalhes sobre o que está errado.
4
Clique em "📲 Enviar no WhatsApp"
O WhatsApp abrirá com a mensagem já formatada. Clique em Enviar para notificar o vendedor responsável pelo pedido.
✅
O pedido não é bloqueado ao reportar uma ocorrência — ele continua normalmente no kanban enquanto o problema é resolvido.
8. Dicas importantes
💡
Sempre preencha NF e Rastreio antes de marcar como Enviado
Essas informações são enviadas automaticamente para o CRM e ficam registradas para consulta futura.
💡
Use a lista de Pré-faturamento para se organizar
Imprima as ordens de faturamento com antecedência para agilizar o processo quando o financeiro liberar.
💡
Não use "Excluir card" para pedidos reais
Use apenas se um card entrou por engano. Para problemas no pedido, use "Reportar Problema" ou "Voltar ao Comercial".
